Логин и пароль панели управления по умолчанию - admin

Для работы CMS необходимо создать базу данных MySQL,
далее перейти по адресу {Ваш домен}/install/
затем указать необходимые данные для соединения с базой и другие параметры приложения

Доступ в панель управления - {Ваш домен}/control/
